Devices¶

class devices.devices.BS600(ser)¶

This class represents the BS600 Balance.

Setup a device as new device or as subdevice.

Assign ser to self._ser and if ser is not already openend (the device is not a sub device) set the in the class defined attributes and open the serial connection. If the connection has to be openend, _ownSer flag is set.

To setup the device as a subdevice simply pass the serial connection to the class constructor. To setup the device as a new device one has to use the openRS232 classmethod.

Parameters:ser (Serial) – Serial connection (pyserial module)

Devicemanager¶

class devices.devicemanager.DeviceManager¶

The DeviceManager manages devices.

It provides methods to add or delete devices, get their most recent measurement values and various other things.

queue = None¶

The queue queues all incoming values (from the opened devices).

They are stored in the format (deviceID, value)

calibrate(rv, calibration, unit='')¶

Return a calibrated version of the value.

Parameters:
  • rv (Value) – Value to be calibrated
  • calibration (Tuple) – Calibration tuple
  • unit (String) – Unit

The calibration tuple must contain either two integers m, c to create a function of style f(x) = m x + c or two tuples containing two points on such a linear function, used to compute m and c.
